Гігієна у фармації

 

  • Робоча програма навчальної дисципліни для спеціальності "226 Фармація, промислова фармація" (5,0 денна ф.н., 5,6 з.ф.н., 4,6 з.ф.н.)
  • Тематичний та календарний план лекційних та практичних занять: 
    • для спеціальності "7.12020101 Фармація" 3 курс (денна форма навчання);
    • для спеціальності "226 Фармація, промислова фармація" 1 курс (денна форма навчання);
    • для спеціальності "226 Фармація, промислова фармація" 1 курс мед 4.0 (денна форма навчання);
    • для спеціальності "226 Фармація, промислова фармація" 1 курс 4.0 (денна форма навчання);
    • для спеціальності "7.12020102 Клінічна фармація" 3 курс (денна форма начання);
    • для спеціальності "226 Фармація, промислова фармація" (факультет з підготовки іноземних громадян);
  • Перелік питаннь для підготовки до підсумкового модульного контролю
  • Перелік рекомендованої літератури
